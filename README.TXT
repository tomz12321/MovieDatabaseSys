USER INSTRUCTIONS:
1) Download the zip package from Github.
2) Unzip the package
3) Click on package.bluej to demo

DEMO INSTRUCTIONS:
After launch to blueJ,
1) Compile
2) Right click Class MovieSystem to new a MovieSystem object
3) Right click on MovieSystem object to execute method start() to operate the system.

New a MovieSystem object from the class MovieSystem, then use the method: start() to operate the system.
Step 1) System will load in database from the file myvideo.txt
Step 2) Press 1 to search movie from the list
Step 3) Press 2 or 3 to add/delete movie into the list.
Step 4) Press 4 to input a rating, the system will display the list of movies which rating is higher than you input..
Step 5) Press 5 to edit the actors and rating of the movie you choose. (HD version)
Step 6) Press 6 to exit the game
